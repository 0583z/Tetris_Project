俄罗斯方块网络对战项目运行与测试说明

一、开发环境

 开发语言

Python 3.9+

开发框架

Pygame

 通信方式

TCP Socket

 数据存储

JSON 文件

推荐运行环境

Windows 10/11
Python 3.9 及以上版本
Pygame 2.x

二、项目目录说明

Tetris_Project
│
├── client.py                 # 客户端程序
├── server.py                 # 服务端程序
├── assets/                   # 图片、音效资源
├── data/                     # 用户及排行榜数据
├── docs/
│   └── RUN_AND_TEST_GUIDE.md
└── README.md

三、环境配置

安装 Python 后执行：
bash
pip install pygame

安装完成后即可运行项目。

四、启动服务端

进入项目目录后执行：
bash
python server.py

服务端负责：

用户登录
用户注册
好友管理
对战邀请
房间创建
游戏数据同步

五、启动客户端

打开新的命令窗口：
bash
python client.py


输入：

服务器IP：
127.0.0.1
用户名：

任意未注册用户名

第一次登录会自动创建账号。

 六、单人模式测试

测试内容包括：

方块生成
左右移动
快速下落
方块旋转
消行判定
得分统计
游戏结束
本地最高分保存

七、双人模式测试

需要开启：

一个服务端
两个客户端

测试流程：

1. 两个客户端登录不同账号
2. 添加好友
3. 发起对战邀请
4. 接受邀请
5. 开始游戏
6. 测试攻击行同步
7. 测试胜负判定
8. 测试断线处理

八、排行榜测试

完成游戏后：

检查：

分数是否写入
排行榜是否刷新
排名是否正确

九、数据文件说明

项目采用 JSON 文件保存：

用户账号
好友关系
排行榜
游戏记录

无需数据库即可运行。

 十、注意事项

1. 服务端必须先启动。
2. 客户端需连接正确服务器 IP。
3. assets 目录不可删除。
4. JSON 数据文件不能为空。
5. 多人联机需连接同一服务端。

十一、测试结果

经过测试：

单人模式运行正常。
双人联机正常。
数据同步正常。
排行榜更新正常。
用户登录及注册正常。
项目运行稳定，无新增功能变化。
